# Calculadora-IMC-Kotlin
Aplicativo Android desenvolvido com Kotlin e Jetpack Compose que calcula o Índice de Massa Corporal (IMC) com base na altura e peso informados pelo usuário.

## Funcionalidades
* Inserção de altura (cm) e peso (kg)
* Cálculo automático do IMC
* Exibição do resultado com:
* Valor do IMC
* Classificação (ex: peso ideal, sobrepeso, obesidade)
* Mudança de cor do resultado conforme a faixa do IMC

## Fórmula utilizada

IMC = peso / (altura * altura)

A altura é convertida de centímetros para metros antes do cálculo.
